# MarsRover

## Purpose
Finding the last position of Mars Rover for processing on the board

## Introduction

There is a Mars Rover on a board. Board have a borders. The rover has a position. When list of processes is entered, rover moves by these directions. Basically user should enter 3 inputs;

<ol>
  <li>Board</li>
  <li>Current Position</li>
  <li>Process List</li>
</ol>

## Limitations
### Board Limitations
Board input should be entered only digits with blanks. Digit count should be 2 and there should be one blank in the digits
### Current Positions Limitations
Current position should be entered with 2 integer, one direction with one blank. Direction should be only "W", "E", "N", "S"
### Process List Limitations
Process list should be entered with only letters. Letter should be only "R", "L", "M". This letter means;
<ol>
  <li>R: spin 90 degrees right</li>
  <li>L: spin 90 degrees left</li>
  <li>M: move forward one grid point, and maintain the same heading</li>
</ol>
